## RutaSmart



Este proyecto consiste en el desarrollo de un sistema enfocado en la gestión de entregas donde se puedan manejar de forma organizada los clientes, pedidos, rutas y repartidores. La idea es tener control completo del proceso de envío, desde que el pedido se registra hasta que se entrega, utilizando estados como pendiente, en ruta y entregado.



Sistema de logística y control de entregas desarrollado en C#.

## Descripción
RutaSmart nace para resolver uno de los problemas más comunes en los procesos de entrega: la falta de organización y control en los pedidos. Muchas veces las empresas manejan la -información de clientes, rutas y entregas de forma desordenada, lo que provoca retrasos, pérdida de información y dificultades para dar seguimiento a los envíos.

Con este sistema se busca centralizar todo el proceso de logística en una sola aplicación, permitiendo registrar clientes, administrar pedidos, asignar rutas y llevar control de los repartidores de manera más eficiente. Además, el sistema permite conocer el estado de cada entrega en tiempo real mediante estados como pendiente, en ruta y entregado, facilitando una mejor gestión y seguimiento de los pedidos.
